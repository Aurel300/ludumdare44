### Your Life is Currency ###

## Mechanics ##

 - shmup
 - player
   - coin-op machine
 - coins
   - life - get to 0 = game over
   - ammo - lose some as you shoot
 - coin slot
   - collect enemy shots into the slot to gain life + ammo
   - get hit elsewhere = lose life
 - shop
   - between levels, buy upgrades

 - enemies
   - also coin-op machines
   - pinball
   - claw game
   - coin on a string
   - one-armed bandit
